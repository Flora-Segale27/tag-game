# Tag Game

A simple interactive browser game built with vanilla JavaScript and HTML5 Canvas.

## How to Run
1. Open `index.html` in your browser.
2. Edit files in `src/` to build your game.

## Structure
- `src/` — JavaScript code  
- `assets/` — images, audio, sprites  
- `docs/` — design notes  
- `tests/` — test files
